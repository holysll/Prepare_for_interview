# Web基础

## 1.常用的网络传输协议
* TCP：传输控制协议，可靠传输，面向连接
* UDP：用户数据包协议，不可靠传输，面向无连接
* FTP：文件传输协议,用于上传和下载文件
* HTTP：超文本传输协议，基于TCP/IP通信协议，面向对象
* SMTP：邮件传输协议
* TELNET：Internet远程登录服务的标准协议和主要方式
* DNS：域名系统，将域名解析为ip地址
